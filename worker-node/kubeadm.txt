i want to add new node. how do i get the worker node join command?                                                                                                                                               
                                                                                                                                                                                                                   
● Run this on the control plane node (192.168.1.248):                                                                                                                                                              
                                                                                                                                                                                                                   
  kubeadm token create --print-join-command                                                                                                                                                                        
                                                                                                                                                                                                                   
  This generates a fresh token and prints the full join command, something like:                                                                                                                                   
                                                                                                                                                                                                                   
  kubeadm join 192.168.1.248:6443 --token <token> --discovery-token-ca-cert-hash sha256:<hash>                                                                                                                     
                                                                                                                                                                                                                   
  Then run that output command on the new worker node (with sudo).                                                                                                                                                 
                                                                                                                                                                                                                   
  ---                                                                                                                                                                                                              
  Note: Tokens expire after 24 hours by default. If you want a longer-lived token:
  kubeadm token create --ttl 0 --print-join-command  # never expires                                                                                                                                               
  kubeadm token create --ttl 48h --print-join-command  # 48 hours
